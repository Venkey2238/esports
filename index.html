Tournament Rules

Rule 1: Shapeshift Kill Restriction

When an Impostor takes a specific player's shapeshift, they can only kill that player.

Example: If Impostor shapeshifts as RODY, only RODY can be killed.

Rule 2: No Other Kills During Shapeshift

While shapeshifted, killing any other player is NOT allowed.

No Body Reports

No body can be reported until the Impostor kills the player they shapeshifted as.

Wrong Kill = Elimination

Wrong kill results in immediate Impostor elimination from the match.

Tournament Format

  • 8 Players (1 Impostor + 7 Crewmates)
  • 8 Rounds - Each player becomes Impostor once
  • Point System as detailed below

Point System

Impostor Points

  • +2 Each correct kill
  • +4 Bonus for winning round (eliminate all)

Crewmate Points

  • +3 Each surviving crewmate
  • Survival requires: Complete tasks OR catch Impostor's wrong kill
